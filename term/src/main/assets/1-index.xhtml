TTerm is a terminal emulator for Android with improved support for on-screen keyboards (also called virtual keyboards, soft keyboards or IMEs) and improved documentation. It has extra keys for `Ctrl´, `Tab´, `Esc´, `/, `-´, the cursor keys and `Fn´ so that you can continue using your favorite on-screen keyboard even if it lacks those keys.

TTerm is based on Jack Palevich's "Terminal Emulator for Android", which I'll abbreviate as TEA.

TTerm is available for Android 4.4+. The native code is compiled for the architectures arm64, armeabi, armeabi-v7a, mips, mips64, x86 and x86_64. Contact me at thorbj.hansen@gmail.com if you have an unsupported architecture.

TTerm is still in beta, so remember to save your work early and often.

If you find bugs or want to help, contact me at thorbj.hansen@gmail.com.
