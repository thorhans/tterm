This package provides a fairly complete VT100 terminal emulator {@link de.t2h.tterm.emulatorview.TermSession TermSession} and a corresponding Android view {@link de.t2h.tterm.emulatorview.EmulatorView EmulatorView}.

Most users will create a TermSession, connect it to an {@link java.io.InputStream InputStream} and {@link java.io.OutputStream OutputStream} from the emulation client, then instantiate the EmulatorView and add it to an activity's layout.
